Demo

在微信上充值话费，早已不是新鲜事，而现在微信新增了一项功能：赠送手机充值卡。功能入口位于：“我的钱包”-“手机充值”中。 你可以选择“以爱之名”和“我们的友谊”两种充值卡，均能充值移动、联通、电信的话费或流量，而且有多款精美卡面可选。其中“以爱之名”：可以充值30元话费，或者充值500MB流量；而“我们的友谊”可以充值20元话费，或者充值30MB流量；充值卡购买后，有效期1年，仅能赠送给好友，若对方24小时未领取，则会退回你的微信卡包，可自己使用。

Usage

调用方式：
Scrollbar({
	id:'scroll', //id名称
	height:80 //可视区域高度
})
属性 备注说明
id string，例如："scrollbar"
height number，内容高度
